Geoconnex Registry

Geologic features are real things having each a set of data authored by multiple sources. The Registry is meant to be a centralized catalogue providing, for each feature, a concise info and links pointing to the available data. The elaboration of this registry represents the main component of the Geoconnex project that aims to make hydrologic data as easily discoverable, accessible, and usable as possible.

The key to find the web resource data of a geologic feature in the Registry is encoded as a URL consisting in a concatenation of the following:

  • Protocol and domain name : “https://geoconnex.ca”
  • Path
    • e.g. “/id/someCategory/instanceName”
  • Parameters string for format
    • e.g. “?f=xml”

An example of a unique data URL for the Richelieu river is : https://geoconnex.ca/gsip/id/waterbody/Richelieu?f=xml
